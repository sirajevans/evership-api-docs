# Rates

This "Get Rates" endpoint allows an admin or system user to retrieve the available checkout delivery options, delivery rates and providers in the request. Evership runs the data against backend carrier systems and matches key services to the provided collection and delivery addresses. {{ className: 'lead' }}

---

## Validate address {{ tag: 'POST', label: 'https://api.evership.app/rates' }}

<Row>
  <Col>
    ## Overview

    Using the “Validate Address” API, you can validate an address without needing to implement and pay for Google Maps address validation.  
    This API consumes the address passed and then returns the address found during the geo-coding process, the validation status, and any partial match suggestion if applicable.

    ## Request

    **Example GET URL:**

    ```
    GET https://api.evership.app/v1/validate-address?address1=Unit 12&address2=113 Loop St&suburb=Cape Town&province=Western Cape&postal_code=8000&country=South Africa
    ```

    ### Request Fields

    <Properties>
      <Property name="address1" type="string">
        The first line of the address, typically including the unit or apartment number.
      </Property>
      <Property name="address2" type="string" required>
        The second line of the address, typically including the street name and number.
      </Property>
      <Property name="suburb" type="string" required>
        The suburb or neighborhood of the address.
      </Property>
      <Property name="province" type="string" required>
        The province or state where the address is located.
      </Property>
      <Property name="postal_code" type="string" required>
        The postal code of the address.
      </Property>
      <Property name="country" type="string" required>
        The country where the address is located.
      </Property>
    </Properties>
  </Col>
